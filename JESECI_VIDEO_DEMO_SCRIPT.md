# Technical Notes for Production

## General Guidelines

**Visual Style:**
- Maintain consistent dark theme throughout (matches platform aesthetic)
- Use platform screenshots for authentic representation
- Add subtle animations and transitions for professional polish
- Ensure text is readable on all background elements
- Use brand colors: Primary accent (teal/blue), secondary accent (purple), neutral backgrounds

**Audio Guidelines:**
- Voiceover should be warm, professional, and approachable
- Avoid monotone delivery—vary pace and emphasis
- Background music should enhance, not distract
- Ensure clear audio levels between voiceover and music

**Pacing:**
- Allow adequate pause time for viewers to absorb visual information
- Match narration pacing to visual complexity
- Keep feature demonstrations concise but clear

## B-Roll Requirements

- Stock footage of diverse learners (various ages, settings)
- Screen recordings of platform features (multiple takes, various scenarios)
- Animated diagrams for Jaclang agentic concepts
- Celebration/confetti animations for achievements
- Progress tracking visualizations

## Accessibility Considerations

- Include closed captions in final video
- Ensure sufficient color contrast in all graphics
- Provide audio description track if needed
- Consider creating shorter versions for social media
